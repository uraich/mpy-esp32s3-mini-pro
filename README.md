# mpy-esp32s3-mini-pro
A collection of MicroPython programs exercising the esp32s3 mini pro peripherals
The esp32s3 mini pro comes with a number of peripherals:
* an ws2812 rgb LED
* 3 push button switches that can be used by projects
* a 128x128 pixel TFT display controlled by a ST7789 display controller
* a IR LED
* a qmi8658 **I**nertial **M**easurement **U**nit (IMU) providing an accelerometer and a gyroscope

This collections of programs demonstrates how to use the devices.
